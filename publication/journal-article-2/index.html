Interactive R App for Carrying Out an Elicitation Process Applied to Prostate Cancer in Colombia.

Andrés Felipe Flórez Rivera, Juan Carlos Correa Morales, Manuel Garcia Flórez
April 2015
PDF

Abstract

In this paper, an interactive R-application for carried out an elicitation process to the vector of parameters of the Multinomial distribution is presented. The application is developed mainly under two R libraries, Shiny and RSQLite. Shiny is a package that allows developing a web application framework for R and RSQLite package embeds the SQLite database engine in R and provides an interface compliant with the DBI package. The application has two main components, the first stores the records of an elicitation, an expert, and a variable. The second components allow the analyst to ask the experts their judgments about previous variables stored, it returns a plot allowing that the expert has feedback about elicited values. The application is tested in the estimation of the prevalence of each level Gleason score in patients who have been diagnosed with prostate cancer in Colombia.
